# Umut Bindebir

Merhaba! Ben Umut. Bilgisayar programcılığı mezunuyum. Yazılıma ve teknolojiye olan ilgim küçük yaşlarda başladı; zamanla hem mobil uygulama geliştirme hem de siber güvenlik alanlarında yeterli yetkinliklere ulaştım. Aktif olarak Linux sistemleri üzerinde çalışıyor, açık kaynaklı projeler geliştiriyor ve bilgi güvenliği üzerine araştırmalar yapıyorum.

- **Mobil Geliştirme:** Flutter & Dart ile Android/iOS için çapraz platform uygulamalar
- **Siber Güvenlik:** Ağ güvenliği, sızma testleri ve zafiyet analizi
- **Linux:** CLI düzeyde Sistem yönetimi, servis yapılandırmaları
- **Web Güvenliği:** OWASP Top 10, SQLi/XSS/IDOR gibi yaygın zafiyet testleri
- **Araçlar:** Wireshark, Nmap, Metasploit, Burp Suite ve daha fazlası..
